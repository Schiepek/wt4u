Wt4u (Working Time for You) ist eine Webapplikation zur Erfassung von Arbeitszeiten und Projektarbeitszeiten. Als Nutzer mit Arbeitgeberrechten können neue Projekte und User erstellt werden, ausserdem können die vorhandenen Daten umfangreich ausgewertet werden.
